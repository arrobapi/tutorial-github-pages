¿Cómo actualizar tu web en el futuro?

Si modificas tu código en el ordenador o la Inteligencia Artificial genera cambios y quieres que se actualicen en vivo:

01

Entra a GitHub

Navega a tu repositorio en GitHub donde guardaste los archivos.

02

Add File > Upload

Presiona el menú de arriba a la derecha y selecciona subir archivos.

03

Sube y Commit

Arrastra tus archivos nuevos (sobrescribirán los viejos) y confirma los cambios.

Alternativa Avanzada: Flujo de Consola con Git

Para el futuro

Cuando tu proyecto crezca o hagas actualizaciones muy frecuentes, te resultará más cómodo utilizar la consola de comandos de Git.

Inicializar y subir por primera vez 
git init
git add .
git commit -m "Mi primer commit"
git branch -M main
git remote add origin https://github.com/TU-USUARIO/TU-REPOSITORIO.git
git push -u origin main
Actualizar cambios más adelante 
git add .
git commit -m "Actualizando contenido de la web"
git push origin main
